Assignment
************

|Assignment|

#. Create a new repository in Github.

#. Create a new branch.

#. Add an XMl file in the branch.

   Think of content that can be organized by semantic meaning.

   For example:

   .. code-block:: XML

     <people>
       <person>
         <age>3</age>
         <sex>M</sex>
         <name preferred="Joe">Joseph</name>
       </person>
     </people>

#. Add data in the file.

   Use at least 4 elements and one attribute.
